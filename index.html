Other ways to get reminded

iOS Shortcuts automation (most reliable):

  1. Open the Shortcuts app → Automation tab → +
  2. Choose Time of Day, set your evening time, Daily, then Run Immediately
  3. Add action → search Open URLs and pick it. (Not “Open App” — installed web apps don’t appear in that list.)
  4. Tap the URL field and enter https://grsr.github.io/diary/

Now your phone opens the diary automatically every evening. Tip: turn off “Notify When Run” if you don’t want the extra banner.

Apple Health

A web app can't read Apple Health directly (only native apps can) — but you can push data in with a quick Shortcut that reads last night's sleep and opens the diary with it already filled in.

Set up the “sleep from Health” Shortcut
  1. Open the Shortcuts app → + to create a new shortcut.
  2. Add action Find Health Samples → type Sleep Analysis; set it to Today (or last night).
  3. Add an action to total the hours asleep into a variable (e.g. Calculate Statistics → Sum, or use the sample’s duration).
  4. Add Open URLs with: https://grsr.github.io/diary/?sleep= then insert your hours variable right after the =.
  5. Run it each morning — or attach it to a Time of Day Automation so your sleep is waiting for you.

The diary also accepts ?date=YYYY-MM-DD (defaults to today) and ?steps= (ticks “exercise” if you were active). It fills the day in and saves automatically.
